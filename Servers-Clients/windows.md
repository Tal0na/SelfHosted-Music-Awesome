# 🖥️ Best Windows Clients for Selfhost Music

A curated list of Windows desktop clients compatible with **Selfhost Music** and other Subsonic-based servers.

---

## ⭐ Recommended Clients

### Feishin

- **API Support:** ✅ Navidrome / ✅ Jellyfin / ✅ OpenSubsonic
- **License:** GPL-3.0
- **Open Source:** ✅ Yes
- **Price:** 🆓 Free
- **Rating:** ⭐⭐⭐⭐⭐
- **Status:** 🟢 Actively maintained
- **Notes:**
  - Modern desktop UI
  - Support for multiple music servers
  - Active development and recent beta releases
  - Cross-platform client

---

### Supersonic

- **Support:** ✅ Selfhost Music / ✅ Subsonic / ✅ OpenSubsonic
- **License:** GPL-3.0
- **Open Source:** ✅ Yes
- **Price:** 🆓 Free
- **Rating:** ⭐⭐⭐⭐☆
- **Status:** 🟢 Actively maintained
- **Notes:**
  - Lightweight desktop client
  - Written in Go
  - Focused on speed and simplicity

---

### Aonsoku

- **API Support:** ✅ Navidrome / ✅ Jellyfin
- **License:** MIT
- **Open Source:** ✅ Yes
- **Price:** 🆓 Free
- **Rating:** ⭐⭐⭐☆☆
- **Status:** 🟢 Actively maintained
- **Notes:** Subsonic-focused client with a highly polished **Spotify-style UI**. 🎵

---

### 🎧 Psysonic

- **API Support:** ✅ Navidrome / ✅ Gonic
- **License:** GNU
- **Open Source:** ✅ Yes
- **Status:** 🟡 Psysonic is under heavy active development
- **Rating:** ⭐⭐⭐☆☆

#### 🚀 Highlights
- Synchronized Lyrics
- Last.fm Integration: Direct scrobbling
- Clean and modern interface
