# 🟣 Jellyfin Themes

Themes and UI customization for Jellyfin.

---

## 🎨 What can be customized?

- Interface themes  
- Custom CSS  
- Layout and components  

---

## 📦 Examples
[jellyfin-themes](https://github.com/derektata/jellyfin-themes)
[THEMES](https://github.com/awesome-jellyfin/awesome-jellyfin/blob/main/THEMES.md)
- Glass UI styles  
- Custom dashboards  

---

## 🛠️ How to use

1. Go to Dashboard → General  
2. Add custom CSS  
3. Apply theme plugins (if available)  

---

## ⚠️ Notes

- Some themes require plugins  
- CSS changes may break after updates
