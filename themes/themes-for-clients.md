# 📱 Client Themes

Themes and customization for music clients and apps.

---

## 🎨 Supported Clients

- Mobile apps  
- Desktop players  
- Web-based clients  

---

## 📦 Examples

- Custom skins  
- Color themes  
- Layout tweaks  

---

## 🛠️ How to use

- Import theme files (if supported)  
- Use built-in theme settings  
- Apply third-party mods  

---

## ⚠️ Notes

- Not all clients support themes  
- Some require unofficial modifications
