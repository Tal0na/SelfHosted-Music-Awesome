# 🎵 Navidrome Themes

Themes and customization options for Navidrome.

---

## 🎨 What can be customized?

- UI colors and layout  
- Custom CSS overrides  
- Player appearance  

---

## 📦 Examples

- Dark mode variations  
- Minimal UI themes  
- Custom dashboards  

---

## 🛠️ How to use

1. Add custom CSS via configuration  
2. Inject styles using reverse proxy (optional)  
3. Use browser extensions for quick testing  

---

## ⚠️ Notes

- Native theme support is limited  
- Most customization is done via CSS
