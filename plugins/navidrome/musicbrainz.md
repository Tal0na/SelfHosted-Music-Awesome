# 🎼 MusicBrainz Metadata Plugin

Repository: https://github.com/navidrome/navidrome-plugin-musicbrainz

## Description

Enhances metadata by retrieving artist, album, and release information from MusicBrainz.

* Artist metadata
* Album metadata
* Release information
* Improved library organization
